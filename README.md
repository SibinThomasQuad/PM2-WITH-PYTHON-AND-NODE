# PM2-WITH-PYTHON-AND-NODE


--------------------------------------------------  INSTALLATION ---------------------------------------------
With yarn:

yarn global add pm2

npm install pm2 -g

------------------------------------------------- RUNNING FILES ----------------------------------------------
to start node in pm2

pm2 start index.js

with watch 

pm2 start start index.js --watch

to run python in pm2

pm2 start api.py --interpreter=python3 --watch
